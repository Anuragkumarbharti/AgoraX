# AgoraX Agent Configuration

## Project: AgoraX (Community Platform)

### Objective
Build a professional, minimal Android app combining Discord + Quora + Clubhouse with creator economy.

### Tech Stack
- **Mobile**: Flutter
- **Backend**: NestJS
- **Database**: PostgreSQL
- **Voice**: ZEGOCLOUD + WebRTC
- **Real-time**: WebSocket
- **Design**: Dark theme, Glassmorphism, Minimal UI
